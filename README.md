## Sobre
Este aplicativo foi desenvolvido com Flutter e tem como objetivo proporcionar uma experiência agradável de reprodução musical. Com uma interface intuitiva e recursos essenciais, ele permite que você curta suas músicas favoritas com praticidade e estilo.

## Funcionalidades
- Visualização de uma lista de músicas disponíveis
- Controle de reprodução: tocar, pausar e alternar entre faixas
- Exibição de detalhes da música, como título, artista e imagem da capa
- Barra de progresso para acompanhar e navegar pela faixa em execução
